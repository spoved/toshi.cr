# toshi

Toshi is a Crystal shard for building clients for JSON based apis.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     toshi:
       github: spoved/toshi.cr
   ```

2. Run `shards install`
